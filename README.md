# Smart Contact Manager (Work in Progress)

## Overview

A full-stack web application to manage contacts efficiently with features like authentication, search, and secure storage.

## Problem Statement

Managing contacts manually is inefficient and lacks searchability, security, and scalability.

## Planned Features

* User authentication (Login/Signup)
* Add, update, delete contacts
* Search and filter contacts
* Profile image upload
* Secure data handling

## Tech Stack

* Backend: Spring Boot
* Frontend: Thymeleaf / React (TBD)
* Database: MySQL

## Current Status

Project planning and initial setup phase.
